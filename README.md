# Meedan UI Guide

The Meedan UI Guide helps Meedanis develop UIs in the browser.

## Install & run

* `git clone https://github.com/meedan/meedan-ui-guide.git`
* `npm install`
* `gulp`

This should start a local web server running at: `http://localhost:3000/`
